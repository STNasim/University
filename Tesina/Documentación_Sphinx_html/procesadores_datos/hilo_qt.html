hilo_qt¶

class procesadores_datos.hilo_qt.HiloProcesamiento(pipeline: Pipeline, reportar_calidad: bool, parte: str)¶

Bases: QThread

Clase que ejecuta el procesamiento de los datos en un hilo.

Signal progreso:

Señal que indica el progreso del procesamiento.

Signal generar_reporte:

Señal que indica que se debe generar un reporte.

Signal termino_parte:

Señal que indica que se terminó una parte del procesamiento.

Signal terminado:

Señal que indica que se terminó el procesamiento.

Signal candado:

Señal que indica si se debe bloquear la interfaz.

run() → None¶

Ejecuta el procesamiento de los datos.

set_parte(parte: str) → None¶

Establece la parte del procesamiento que se debe ejecutar.

Parámetros:

parte (str) – Indica la parte del procesamiento que se debe ejecutar.
