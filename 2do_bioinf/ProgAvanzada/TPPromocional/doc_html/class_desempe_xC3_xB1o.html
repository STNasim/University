Descripción detallada

La clase se encarga de graficar una función de desempeño mediante puntos.

La gráfica mostrada cuenta de ejes, una grilla y un fondo con un gradiente gris a blanco. Los puntos se grafican mediante discos naranjas con bordes azules. Se representan tambien, rectas que representan el valor máximo, mínimo y el valor medio de los puntos cargados. Finalmente, tambien permite seleccionar si se quiere unir los puntos mediante lineas o mediante una función interpolada.
