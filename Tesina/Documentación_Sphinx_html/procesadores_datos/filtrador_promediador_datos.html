Clase que filtra los datos de los genes por error estándar.

static filtrar_df_por_error_estandar(df: DataFrame, valor_filtro: float, agregar_info_adicional: bool = False) → Tuple[DataFrame, DataFrame]¶

Filtra los datos de los genes por error estándar.

Parámetros:

  • df (pd.DataFrame) – DataFrame con los datos de los genes.

  • valor_filtro (float) – Valor del error estándar para filtrar los datos.

  • agregar_info_adicional (bool, optional) – Indica si se agregan columnas adicionales al DataFrame de los datos que no pasaron el filtro, defaults to False.

Devuelve:

DataFrame con los datos que pasaron el filtro, DataFrame con los datos que no pasaron el filtro.

Tipo del valor devuelto:

Tuple[pd.DataFrame, pd.DataFrame]

static filtrar_y_juntar_dict_df(datos_por_grupo: dict, error: float, id: str) → Tuple[DataFrame, List, Dict]¶

Filtra los datos de los genes por error estándar y los junta en un solo DataFrame.

Parámetros:

  • datos_por_grupo (dict) – Diccionario con los datos de los genes agrupados por grupo.

  • error (float) – Valor del error estándar para filtrar los datos.

  • id (str) – Nombre de la columna que contiene el identificador de los genes.

Devuelve:

DataFrame con los datos que pasaron el filtro, lista con los grupos, diccionario con los datos que no pasaron el filtro.

Tipo del valor devuelto:

Tuple[pd.DataFrame, List, Dict]
