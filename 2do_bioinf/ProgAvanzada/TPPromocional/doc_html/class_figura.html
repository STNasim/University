Descripción detallada

Clase abstracta que se encarga de generalizar metodos y atributos para las clases derivadas.

La clase tiene un constructor para establecer el objeto Pixel y un metodo dibujar.

Precondición
La clase es utilizable si se necesita graficar variedad de figuras distintas y se quiera generalizar ciertos aspectos comunes a las figuras, en este caso establecer su color y dibujarlas.
