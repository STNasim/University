Los materiales que necesito para mi hobby

En esta página voy a mostrar los materiales que necesito para practicar mi hobby: hacking.

Estos son algunos de los materiales que uso:

  • Un ordenador con conexión a internet
  • Un sistema operativo adecuado (como Linux o Windows)
  • Software especéfico para realizar auditoréas de seguridad (como Nessus, OpenVas, w3af, nexpose o nikto)
  • Herramientas para analizar y explotar vulnerabilidades en servidores web (como whatweb, cmscan, joomscan, droopescan o wpscan)
